# Projeto Notas :memo:

## Utilidade:

Neste projeto é possível criar, salvar e editar notas.

## Tecnologias uilizadas:

- Javascript;
- Html;
- CSS;
- Node.js
- Express;
- Handlebars;
- MongoDB;
- Nodemon.

## CRUD completo usando rotas para:

- [x] Criação da nota;
- [x] Visualização de todas as notas;
- [x] Visualização de detalhes de uma única nota;
- [x] Editar notas;
- [x] Deletar notas.
